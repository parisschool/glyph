# Glyph-Viz 

Una librería de Python para visualizar estructuras de datos (Heaps, AVL, Stacks, Queues, etc.).

## Características Principales

* **Abstracción Completa**: No necesitas configurar nodos o aristas manualmente; la librería se encarga de recorrer la estructura por ti.
* **Soporte Multiestructura**: Visualiza desde estructuras lineales simples hasta árboles balanceados y filtros probabilísticos.
* **Integración con Colab**: Diseñada para funcionar sin problemas en cuadernos interactivos.
* **Estilo Personalizado**: Incluye utilerías para dar color y formato a las salidas en consola.

## Requisito previo
Esta librería utiliza `graphviz`. Asegúrate de tenerlo instalado en tu sistema:
- **macOS**: `brew install graphviz`
- **Linux**: `sudo apt-get install graphviz`

## Instalación
```bash
pip install glyph-viz
```
